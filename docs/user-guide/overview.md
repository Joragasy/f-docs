# Overview

f-docs is a documentation framework built with MkDocs and Material theme.

## Key Features

- **Fast Static Site Generation**: Generate blazing-fast static sites
- **Beautiful Material Design**: Modern, responsive design with dark mode
- **Full-Text Search**: Built-in search functionality
- **Code Highlighting**: Syntax highlighting for multiple languages
- **Mobile Responsive**: Looks great on all devices

## Architecture

```
f-docs
├── mkdocs.yml          # Configuration file
├── docs/               # Documentation source files
│   ├── index.md       # Home page
│   ├── getting-started/
│   ├── user-guide/
│   └── api/
└── site/              # Generated static site
```

## How It Works

1. Write documentation in Markdown
2. Configure navigation in `mkdocs.yml`
3. MkDocs generates static HTML
4. Deploy to any hosting provider
